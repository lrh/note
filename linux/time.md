
# Linux下时间命令：

```bash

# UNIX时间戳;
date +%s
# > 1563267847

# 日期字符串 => 时间戳
date -d "2015-08-04 00:00:00" +%s
# > 1438617600

# 时间戳 => 日期字符串
date -d @1438617600  "+%Y-%m-%d %H:%M:%S"
# > 输出：2015-08-04 00:00:00

# 如果需要得到指定日期的前后几天：
seconds=`date -d "2015-08-04 00:00:00" +%s`     # 得到时间戳
seconds_new=`expr $seconds + 86400`             # 加上一天的秒数86400
date_new=`date -d @$seconds_new "+%Y-%m-%d"`    # 获得指定日前加上一天的日前

```
